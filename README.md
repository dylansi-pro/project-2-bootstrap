# Koko Café - Brand Website with Bootstrap

Ce projet consiste en la création d'un site vitrine complet pour **Koko Café**, un café de spécialité proposant des pâtisseries artisanales et des brunchs dans une ambiance chaleureuse.

## Évolution du projet (Refonte Bootstrap)

Ce projet a été initialement conçu en CSS pur avant d'être migré vers **Bootstrap 5**. Cette refonte a permis :

- **Une meilleure réactivité** grâce au système de grille et aux composants natifs de Bootstrap.
- **Une maintenance simplifiée** tout en conservant une identité visuelle unique grâce à une feuille de style CSS personnalisée (`style.css`) qui surcharge les styles par défaut pour préserver l'identité de marque du café.

## Fonctionnalités

- **Responsive Design :** Adaptabilité totale sur mobile, tablette et ordinateur.
- **Performance :** Utilisation efficace de Bootstrap pour la mise en page tout en conservant un design sur-mesure.

## Structure des fichiers

```text
project-2-bootstrap/
├── css/
│   └── style.css       # Styles personnalisés conservant l'identité de Koko Café
├── images/             # Ressources visuelles
├── index.html          # Page d'accueil
├── about.html          # Page de présentation / Staff
├── menu.html           # Carte détaillée
├── gallery.html        # Galerie photo
└── contact.html        # Formulaire de contact
```
